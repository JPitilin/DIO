## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por JÚLIO CESAR RICHARD PITILIN. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serÃ£o descritas as etapas do projeto:

Etapa 1:

Nome da ferramenta: Amazon S3 Intelligent-Tiering.
Foco da ferramenta: Redução de custos com armazenamento.
Descrição de caso de uso:
O Amazon S3 é um serviço usado para guardar arquivos, como documentos, imagens e backups. O Intelligent-Tiering analisa automaticamente com que frequência esses arquivos são acessados. Arquivos muito usados ficam em uma área rápida e arquivos pouco usados são movidos para uma área mais barata, sem necessidade de intervenção humana.


Etapa 2:

Nome da ferramenta: Amazon EC2 Spot Instances.
Foco da ferramenta: Redução de custos com servidores.
Descrição de caso de uso: 
As Spot Instances permitem usar servidores EC2 que estão ociosos na AWS com descontos que podem chegar a até 90% em relação ao preço normal.
Elas são ideais para tarefas que não precisam ficar ligadas o tempo todo, como processamento de dados, testes ou análises.


Etapa 3:

Nome da ferramenta: AWS Cost Explorer.
Foco da ferramenta: Controle e análise de gastos.
Descrição de caso de uso:
O AWS Cost Explorer permite visualizar de forma simples onde o dinheiro está sendo gasto na AWS. Ele mostra gráficos, relatórios e tendências de consumo.
Com essas informações, a empresa consegue identificar serviços caros ou mal utilizados e tomar decisões rápidas para reduzir custos.

## Conclusão
A implementação de ferramentas na empresa Abstergo Industries tem como esperado redução de custos, evita dispercios e melhora o controle financeiro do ambiente na nuvem.
